# Proyecto e-Mercado JAP — Caso de Estudio QA

## Descripción general

Caso de estudio de testing funcional, de seguridad y de regresión aplicado
a e-Mercado, una plataforma de e-commerce desarrollada en el marco del
programa Jóvenes a Programar (JAP) — Plan Ceibal, Uruguay.

El proyecto simuló un ciclo completo de aseguramiento de calidad en 4 etapas:
planificación, ejecución de pruebas, retesting y evaluación final.

## Contexto

- **Programa:** Jóvenes a Programar — Nivel 1, Fase 2
- **Equipo:** Grupo 280 — Equipo 4
- **Año:** 2023
- **Rol de Sharon Pereira:** Coordinadora de Retroalimentación / Líder de Documentación

## Alcance de las pruebas

| Módulo | Tipo de prueba |
|---|---|
| Registro e inicio/cierre de sesión | Funcional, seguridad, casos borde |
| Perfil de usuario | Funcional, persistencia de datos |
| Búsqueda y categorías de productos | Funcional, regresión |
| Carrito de compras | Funcional, flujo completo |
| Proceso de pago y tipo de envío | Funcional, casos negativos |
| Base de datos | Validación con SQL |
| APIs | Solicitudes GET, POST, PUT y DELETE |

## Etapas del proyecto

| Etapa | Descripción |
|---|---|
| Etapa 1 | Planificación y diseño de casos de prueba |
| Etapa 2 | Ejecución de pruebas — 81 CP diseñados sobre 27 CU |
| Etapa 3 | Retesting y pruebas de confirmación sobre versión 2.0 |
| Etapa 4 | Evaluación final e informe ejecutivo |

## Resultados destacados — Etapa 4

- Cobertura de pruebas: 100% (funcional, seguridad, rendimiento, usabilidad, APIs)
- 27 defectos identificados en total; 7 corregidos y verificados satisfactoriamente
- Tiempo de respuesta promedio: por debajo de 1 segundo
- Reducción de vulnerabilidades críticas: 10% respecto al inicio del proyecto
- Defecto crítico sin resolver: eliminación de la opción de compra, bloqueando la funcionalidad principal del sistema

## Herramientas utilizadas

- Gestión de incidencias: Trello
- Diseño y ejecución de casos de prueba: Google Sheets
- Validación de base de datos: SQL (MySQL)
- Pruebas de API: solicitudes GET, POST, PUT y DELETE
- Plataformas de prueba: Windows 10/11, OS X, Android 11.0
- Navegadores: Chrome, Edge, Firefox, Opera, Safari
- Control de versiones: Git / GitHub
