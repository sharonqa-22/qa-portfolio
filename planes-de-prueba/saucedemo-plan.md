## Objetivo

Verificar el correcto funcionamiento de los modulos principales de SauceDemo: autenticacion, catalogo de productos, carrito de compras y proceso de checkout.

| Modulo | Incluido |
|---|---|

| Catalogo de productos | Si |
| Carrito de compras | Si |
| Checkout | Si |
| Logout | Si |

## Tipos de prueba

- Pruebas funcionales
- Pruebas de regresion
- Pruebas de flujo end-to-end

## Criterios de entrada

- La aplicacion esta disponible y accesible
- Las credenciales de prueba estan activas
- El navegador Chrome esta actualizado

## Criterios de salida

- Todos los casos de prueba fueron ejecutados
- Los bugs encontrados fueron documentados
- No quedan bugs de severidad Alta o Critica sin reportar
